Swipe-Server
============

A very barebones node server for keeping track of buy and sell orders in the Case Swipe system.

Public API
----------

GET /buy

ex.
```javascript
[
  {
    id: 1,
    name: "will",
    price: 2.45
  }, {
    id: 2,
    name: "clarence",
    price: 7
  }, {
    id: 3,
    name: "pete",
    price: 3.5
  }
]
```
